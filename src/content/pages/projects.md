- **[Literal](https://github.com/doedja/literal)** — A simple blogging platform built with Astro. Powers this blog.
- **[Scenecut](https://github.com/doedja/scenecut)** — Cross-platform keyframe generation alternative for scene detection, mainly for anime.
- **[KBBI](https://kbbi.doedja.com)** — Indonesian language dictionary with modern features like instant search.
- **[Ejaan](https://ejaan.doedja.com)** — Fast and accurate Indonesian spelling checker based on KBBI.
- **[Tesaurus](https://tesaurus.doedja.com)** — Indonesian thesaurus with synonyms, antonyms, and peribahasa/idioms.
- **[Aegisubs Automation Scripts](https://github.com/doedja/aegisub-scripts)** — Lua scripts for Aegisub to help with subtitle editing.
- **Noure** — A platform enabling users to create customized payment profiles for easy sharing.
- **[Yukimedia](https://yukimedia.moe)** — A landing page developed for Yukimedia Indonesia.
- **[JFinder](https://jfinder.doedja.com)** — AI-powered research paper finder and downloader with gap analysis.
- **[Forja](https://forja.id)** — A product studio that builds practical, useful software.
